## big_number

**big_number - это библиотека на C++ для работы с большими целыми числами. Она предоставляет класс `BN` для выполнения арифметических операций с числами произвольной длины, а также реализации различных алгоритмов теории чисел**

## Описание
- Арифметические операции: сложение, вычитание, умножение, деление, возведение в степень
- Работу с системами счисления (десятичная и шестнадцатеричная)
- Проверку простоты (тест Миллера-Рабина)
- Факторизацию чисел (методы Ферма, Полларда P-1 и P-ρ)
- Дискретное логарифмирование (алгоритмы Гельфонда и Полларда)
- Операции по модулю
